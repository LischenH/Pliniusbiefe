# Plinius: Memory Raid 🏛️⚔️

Ein **gamifiziertes Latein-Lernprogramm** für die mündliche Prüfung über den Vesuvbrief.

## 🚀 Installation

```bash
pip install PySide6
python main.py
```

## 🎮 Game Modes

| Modus | Beschreibung | Dauer |
|-------|-------------|-------|
| ⚡ **Echo Mode** | Satz merken → tippen | 60-90 Sek |
| 🎯 **Konstruktionsjagd** | Konstruktionen im Text erkennen | 45-60 Sek |
| 🏃 **Translation Speedrun** | Übersetzen gegen die Uhr | 60 Sek |
| 💀 **Boss Fight** | Vollständige Analyse | 90 Sek |

## 🏆 Progression

- 10 Dungeons (Teil I–X)
- Freischalten nur bei ≥ 70% Score
- XP + Combo-Multiplikatoren
- Schwächen-Tracking

## 📋 Requirements

- Python 3.9+
- PySide6
