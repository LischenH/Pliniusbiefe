# 🏛 PLINIUS: MEMORY RAID

> **12 Tage. 10 Texte. Mündliche Prüfung besiegt.**

Ein ADHS-optimiertes, gamifiziertes Lateinlernprogramm für den Vesuvbrief.

---

## 🚀 Start

### Windows
- Doppelklick auf **`start.bat`**  
  oder direkt `index.html` im Browser öffnen

### Android / iOS
- Datei auf Google Drive / OneDrive hochladen  
- Im mobilen Browser öffnen  
- Oder: Lokal via USB übertragen und im Browser öffnen

---

## 🎮 Spielmodi

| Modus | Beschreibung | XP |
|---|---|---|
| 🧠 **Echo Mode** | Satz 5s lesen → auswendig tippen. Aktives Recall. | 50 XP |
| 🎯 **Konstrukt Jagd** | Markierte Konstruktion identifizieren (Multiple Choice). | 40 XP |
| ⚡ **Übersetzung Speedrun** | 60 Sekunden, so viele Sätze wie möglich übersetzen. | 60 XP |
| 💀 **Bossfight** | Übersetzung + Konstruktion + Stilmittel in 3 Runden. | 100 XP |

---

## 🗺 Progression

- **10 Dungeons** = 10 Teile des Vesuvbriefs
- Dungeon wird **freigeschaltet** wenn vorheriger ≥ 70% Durchschnitt
- **XP-System**: Richtige Antworten + Combo-Multiplikator
- **Schwachstellen-Tracker**: zeigt, welche Konstruktionen du immer wieder falsch machst

---

## 🧠 ADHS-optimiert

- ✅ Sessions 15–25 Minuten
- ✅ Sofortiges Feedback
- ✅ XP + Combo bei jeder richtigen Antwort
- ✅ Sichtbarer Fortschrittsbalken
- ✅ Kein langer Lesemodus — nur aktive Aufgaben

```
index.html         ← KOMPLETT STANDALONE APP (hier starten!)
```
